# Siege Ammunition Tokens (`[ITEM_SIEGEAMMO]`)

Siege ammunition is used by ballistae and other massive engines.

## Core Tokens
- `[ITEM_SIEGEAMMO:ID]` - Starts a new siege ammo definition.
- `[NAME:singular:plural]` - Display name.
- `[CLASS:type]` - Type of engine that fires it (`BALLISTA` is presently the only permitted value)
