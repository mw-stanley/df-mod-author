# Food Tokens (`[ITEM_FOOD]`)

Food tokens define prepared meal types in Dwarf Fortress.

## Core Tokens
- `[ITEM_FOOD:ID]` - Starts a new food definition.
- `[NAME:singular:plural]` - Display name of the meal.
- `[LEVEL:n]` - Complexity/Quality level based on ingredients:
    - `2`: Easy (Default).
    - `3`: Fine.
    - `4`: Lavish.
